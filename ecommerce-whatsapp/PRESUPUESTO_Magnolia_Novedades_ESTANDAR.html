Resumen ejecutivo

Implementación profesional de Facebook Conversion API (CAPI) para seguimiento de conversiones y optimización de campañas de Facebook Ads.

Precio propuesto (Paquete Estándar): ARS 150.000

Alcance del servicio

  1. Configuración en Facebook Business: Pixel, Access Token y validación en Events Manager.
  2. Implementación técnica en el sitio:
    • Variables de entorno (.env): VITE_FACEBOOK_PIXEL_ID, VITE_FACEBOOK_ACCESS_TOKEN.
    • Servicio central facebookService.js para envío de eventos.
    • Integración en: ProductDetail, CartContext, Cart, Registro.
    • Hashing (SHA-256) de email para mejor matching.
  3. Testing y verificación: pruebas en Events Manager y debug.
  4. Documentación: guía técnica y pasos para renovación de token.

Entregables
